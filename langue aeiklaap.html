I – Phonologie & morphophonologie

Voici l’inventaire phonémique de la langue Aeiklaap (/aɛ̝.’jik.laːp/), Aei, étant le nom du peuple parlant la langue, klaap, le nom pouvant être traduit par « langue », possédant dix voyelles pour vingt-sept consonnes

L'écriture de cette langue se fait selon deux systèmes, le premier étant l'écriture romanisée, s'écrivant phonétiquement, selon les normes orthographiques suivantes:

  • /ɛ/ est noté e
  • /ɨ/ est noté i
  • /ɔ/ est noté ö
  • /y/ est noté ü
  • /ʔ/ est noté ‘
  • /θ/ est noté th
  • /ɬ/ est noté hl
  • /ʂ/ est noté sh
  • /ʐ/ est noté zh
  • /j/ est noté y
  • Une voyelle longue est annotée par un redoublement graphique Le deuxième système d'écriture de la langue Aeiklaap sera détaillé dans [[Écriture Aeiklaap]].

Les clusters consonantiques sont acceptés dans les cas ci-dessous :

  • Une occlusive ne peut pas être suivie par une deuxième occlusive.
  • Une nasale ne peut être suivie que d’une deuxième nasale.
  • Les clusters consonantiques doivent suivre une harmonie de voisement.

Les voyelles sont soumises à un système de longueur vocalique et d’affaiblissement, de plus, la deuxième voyelle d’une diphtongue sera toujours affaiblie. Les voyelles centrales (/ə/, /ɨ/), sont les versions non accentuées de /e/ et /i/. Ainsi, en syllabe accentuée où se trouvent /ə/ et /ɨ/, alors, elles se retrouvent décentralisées. L’accent tonique est pénultième. Cela signifie qu’il apparaît sur l’avant-dernière syllabe de chaque mot.

Une syllabe en Aeiklaap suit la structure suivante : (C)(C)(V)V(V)(C), où les diphtongues sont acceptées en position nucléaire. Toutes les consonnes, sauf les glottales et les latérales peuvent occuper la place de la coda.

Le phonème /ʔ/ apparaît entre deux voyelles appartenant à deux voyelles distinctes. Dans ce cas, /ʔ/ sépare les deux syllabes :

  • /‘ae.ɨ/ → /‘ae.ʔɨ/

    II – Morphologie

    L’Aeiklaap est une langue isolante. Les mots se construisent autour de l’assemblage d’au maximum trois concepts, séparés par la syllabe ‘öy (/’ʔɔj/).
